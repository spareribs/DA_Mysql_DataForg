# 题目
1. 查询在 SC 表存在成绩的学生信息



# 解释

明确需要查询的数据表：学生表（`Student`）和成绩表（`SC`）

- 方法一

首先获取到所有存在成绩的学生SId信息

```mysql
mysql> SELECT SId FROM SC GROUP BY SId;

| SId  |
+------+
| 01   |
| 02   |
| 03   |
| 04   |
| 05   |
| 06   |
| 07   |
+------+
```

然后使用条件判断过滤出对应的学生信息


```myqsl
mysql> SELECT * FROM Student WHERE SId IN (SELECT SId FROM SC GROUP BY SId);
```
