# 题目
21. 查询任何一门课程成绩在 70 分以上的姓名、课程名称和分数



# 解释

又是一道可以用万能的方法做完的题目，笛卡儿积加入条件判断

```mysql
mysql> SELECT Student.Sname, Course.Cname, SC.score FROM Student, SC, Course WHERE SC.score >=70 AND Student.SId = SC.SId AND SC.CId = Course.CId;
```
