加入前3名的过滤条件

```mysql
mysql> SELECT *  FROM SC  WHERE ( SELECT count( * ) FROM SC AS biao1 WHERE SC.CId = biao1.CId AND SC.score < biao1.score ) < 2 ORDER BY CId ASC, SC.score DESC;
```
